Pseudo_classes...!

The CSS pseudo classes allow us to style the dynamic states an element such as hover,active and focus state,as well as element that are existing in the document,tree but cant be targeted via the use of other selector without adding any IDS or classes of them, for example, targeting first or last child elements.

A pseudo-class starts with a colon (:) .its syntax can be given with:

Selector:pseudo-class{property:value;}
